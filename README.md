### Installation

~~~
HM-Net_ROOT=/path/to/clone/HM-Net_ROOT
git clone https://github.com/HakkiMotorcu/HM-Net_WAMI $HM-Net_ROOT
cd HM-Net_ROOT
conda create --name <env> --file requirements.txt
conda activate <env>
~~~
